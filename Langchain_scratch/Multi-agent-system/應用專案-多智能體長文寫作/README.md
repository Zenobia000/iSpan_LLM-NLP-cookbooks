# 多智能體長文寫作系統 - 教學課程

## 🎯 課程目標
用最簡單的方式，學會建立 AI 寫作系統

## 誰適合這門課？
- 想了解 AI 如何寫作的人
- 想建立自己的寫作助手的人
- 不需要深厚程式背景，只要會基礎 Python 即可

---

## 📚 課程架構

```
長文寫作系統開發課程
├── 📁 01_概念理解 (30分鐘)
│   ├── 01_什麼是智能體.md - 用生活例子理解 AI 助手

│   ├── 02_建立研究智能體.ipynb - 創建第一個智能體
│   ├── 03_建立寫作智能體.ipynb (選修)
│   └── 04_完整流程.ipynb (選修)
│
└── 📁 03_框架實作 (30分鐘)
    ├── 01_使用CrewAI快速實作.ipynb - 5分鐘建立寫作團隊
    ├── 02_CrewAI長文寫作系統.ipynb (選修)
    └── 03_實戰專案.ipynb (選修)
```

## 📊 學習建議與技巧

### ✅ 建議做法
- **循序漸進**：先概念→再實作→最後框架
- **動手實作**：看懂不等於會做，要實際操作
- **從錯誤學習**：錯誤訊息是最好的老師
- **保持好奇**：試著修改參數看結果

### ❌ 避免做法
- **跳過基礎**：不要直接看框架實作
- **完美主義**：先求能動，再求完美
- **死記程式**：理解概念比背程式重要
- **單打獨鬥**：遇到問題要查資料或問人

### 🔧 除錯技巧
1. **API 錯誤**：檢查金鑰和網路連線
2. **套件問題**：確認版本相容性
3. **輸出異常**：調整 temperature 參數
4. **速度太慢**：減少 API 呼叫次數
